------------------

- Add support for Plone 5

0.7.1 (2015-01-26)
------------------

  [datakurre]

0.6.1 (2014-08-05)
------------------

- Fix issue where language bindings are not set for task queue requests,

  [datakurre]

0.6.0 (2014-05-19)
------------------

- Add taskqueue.add to return task id, which later matches
  request.getHeader('X-Task-Id')
  [datakurre]

0.5.1 (2014-05-14)

  [datakurre]

0.5.0 (2014-04-03)
